# Weather-Bolt-V2

A simple **`weather forecast`** website.

**Tech stack used:**

Front-end: **`Html`**, **`Tailwindcss`**, **`Css`**, **`Js`**, **`hbs`**

Back-end: **`NodeJS`**

API used: **`Mapbox`**, **`OpenWeatherMap`**
